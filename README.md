# Бот-перехватчик заказов для Telegram

Бот читает сообщения **только из указанных чатов** и отправляет подходящие сообщения в одну целевую группу.

## 1) Установка

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 2) Настройка

1. Скопируйте `.env.example` в `.env`.
2. Заполните:
   - `BOT_TOKEN` — токен от `@BotFather`.
   - `SOURCE_CHAT_IDS` — ID чатов-источников через запятую.
   - `TARGET_CHAT_ID` — ID группы-приемника.
   - `ORDER_KEYWORDS` — ключевые слова для поиска заказов (или пусто, чтобы слать все).

## 3) Запуск

```powershell
python bot.py
```

## Ошибка `401 Unauthorized` / `InvalidToken`

Telegram не принял `BOT_TOKEN`. Чаще всего: токен отозван, скопирован с ошибкой или в `.env` лишние кавычки/пробелы.

1. Откройте `@BotFather` → ваш бот → **/token** (или **/revoke**, затем выдайте новый токен).
2. Вставьте токен в `.env` в одну строку, **без** кавычек: `BOT_TOKEN=123456:ABC...`
3. Если токен светился в логах или в чате — считайте его скомпрометированным и **обязательно перевыпустите**.

## Ошибка `TimedOut` / `ConnectTimeout`

Бот ходит на `api.telegram.org`. Если соединение режется или очень медленное, запрос не успевает за стандартное время.

1. Проверьте интернет и VPN (в некоторых сетях API Telegram недоступен без VPN).
2. В `.env` можно увеличить таймауты (секунды):  
   `TELEGRAM_CONNECT_TIMEOUT`, `TELEGRAM_READ_TIMEOUT`, `TELEGRAM_WRITE_TIMEOUT`, `TELEGRAM_POOL_TIMEOUT`.
3. Если нужен прокси: `TELEGRAM_PROXY=http://127.0.0.1:7890` (порт подставьте свой).  
   Для `socks5://` установите зависимость:  
   `pip install "python-telegram-bot[socks]"`.

## Важные условия работы

- Бот видит сообщения только в тех чатах, где он состоит.
- Для групп/супергрупп лучше дать боту права администратора.
- В `@BotFather` отключите Privacy Mode (`/setprivacy -> Disable`), иначе бот не увидит часть сообщений в группах.
- Для каналов бот должен быть добавлен в канал (обычно как администратор).

## Как узнать ID чатов

Простой способ:
1. Добавьте бота `@userinfobot` или любой аналог в нужный чат.
2. Получите `chat_id` (для групп обычно начинается с `-100...`).
3. Вставьте эти ID в `.env`.
