# Как создать Telegram-бота и заполнить `.env`

Ниже пошагово для полей из вашего `.env.example`:

- `BOT_TOKEN`
- `SOURCE_CHAT_IDS`
- `TARGET_CHAT_ID`
- `ORDER_KEYWORDS`

## 1) Создать бота и получить `BOT_TOKEN`

1. Откройте Telegram и найдите `@BotFather`.
2. Нажмите **Start** и отправьте команду `/newbot`.
3. Придумайте:
   - имя бота (любое),
   - username (должен заканчиваться на `bot`, например `order_sniffer_helper_bot`).
4. `@BotFather` пришлет токен вида:
   `123456789:AA...`
5. Вставьте его в `.env`:

```env
BOT_TOKEN=123456789:AAxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## 2) Отключить Privacy Mode (важно для групп)

Иначе бот видит не все сообщения.

1. В `@BotFather` отправьте `/setprivacy`.
2. Выберите вашего бота.
3. Нажмите **Disable**.

## 3) Добавить бота в нужные чаты

Нужно добавить бота:
- во все чаты-источники (где ищем заказы),
- в целевую группу (куда отправляем найденное).

Рекомендуется выдать боту права администратора в группах/каналах, чтобы не было ограничений доступа.

## 4) Получить `chat_id` для `SOURCE_CHAT_IDS` и `TARGET_CHAT_ID`

### Вариант A (самый простой): через `@userinfobot`

1. Добавьте `@userinfobot` в нужный чат.
2. Откройте чат и отправьте любое сообщение.
3. `@userinfobot` покажет `chat id`.
4. Скопируйте ID.

Обычно:
- у групп/супергрупп ID начинается с `-100...`,
- у личных чатов ID обычно положительный.

### Что куда вставлять

- Все ID чатов-источников через запятую:

```env
SOURCE_CHAT_IDS=-1001111111111,-1002222222222,-1003333333333
```

- Один ID целевой группы:

```env
TARGET_CHAT_ID=-1009999999999
```

## 5) Заполнить `ORDER_KEYWORDS`

Это слова-триггеры, по которым бот будет перехватывать сообщения.

Пример:

```env
ORDER_KEYWORDS=заказ,ищу,нужно,срочно,бюджет,тз,проект
```

Если оставить пустым:

```env
ORDER_KEYWORDS=
```

бот будет пересылать **все** сообщения из `SOURCE_CHAT_IDS`.

## 6) Пример полностью заполненного `.env`

```env
BOT_TOKEN=123456789:AAxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
SOURCE_CHAT_IDS=-1001111111111,-1002222222222
TARGET_CHAT_ID=-1009999999999
ORDER_KEYWORDS=заказ,ищу,нужно,срочно,бюджет
```

## 7) Запуск

Запускайте файлом `run.bat`.

Если бот не пересылает сообщения:
- проверьте, что бот реально добавлен в источники и целевую группу;
- проверьте, что `chat_id` без ошибок;
- проверьте, что отключен Privacy Mode;
- проверьте, что в источниках есть сообщения с ключевыми словами (или что `ORDER_KEYWORDS` пустой).
